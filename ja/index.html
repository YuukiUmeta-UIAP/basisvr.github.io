BasisVR on GitHub

クリエイターファースト、クリエイティブフリーダム

BasisはVRゲームのセットアップを簡単にします。
オープンソース、MITライセンスで、クリエイター向けに設計されています。ネットワーク、ユーザー入力、ユーザープレゼンスのための強力なシステムを備えています。

GitHub 特徴 ドキュメント ファンディング 理念

BasisはVRプロジェクトの基盤です

VRアプリケーションのアイデアはあるものの、VR対応に必要な膨大な作業量に圧倒されているなら、もう探す必要はありません。 基本的な部分はすべて網羅します: XR入力、オーディオ、プレイヤーコントローラー、ネットワーク、アセットバンドルの読み込みなど。

サステナビリティとガバナンス

Basisは、コミュニティからの寄付と、フレームワーク上で製品を開発する企業とのコラボレーションによって運営されています。 複数の組織からの資金を集約することで、誰もが恩恵を受けられる困難な課題に取り組んでいます。例えば、揺れシステムとQUIC統合の統合です。
